# Dataset Info

**Name:** Credit Card Customer Data
**Source:** [Kaggle - arjunbhasin2013/ccdata](https://www.kaggle.com/datasets/arjunbhasin2013/ccdata)
**File:** `CC GENERAL.csv`

## How to use
1. Download the CSV from the Kaggle link above (free Kaggle account required)
2. Place it in this `data/` folder, or load directly into your SQL environment

## Why it's not committed directly
This raw dataset is sourced from a public Kaggle dataset and is linked here rather than duplicated in this repo, in line with standard data project practice.

## Columns (summary)
| Column | Description |
|---|---|
| CUST_ID | Unique customer identifier |
| BALANCE | Current account balance |
| BALANCE_FREQUENCY | How frequently balance is updated |
| PURCHASES | Total purchase amount |
| ONEOFF_PURCHASES | One-time purchase amount |
| INSTALLMENTS_PURCHASES | Installment purchase amount |
| CASH_ADVANCE | Cash advance amount taken |
| PURCHASES_FREQUENCY | Frequency of purchases |
| CASH_ADVANCE_FREQUENCY | Frequency of cash advances |
| CREDIT_LIMIT | Credit limit on the card |
| PAYMENTS | Total payments made |
| MINIMUM_PAYMENTS | Minimum payments made |
| PRC_FULL_PAYMENT | % of full payment paid |
| TENURE | Tenure of credit card service (months) |
